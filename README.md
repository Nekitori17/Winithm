## 🤝 Contribute

Winithm is built by the community, for the community! We'd love to have your help. There are two primary ways you can get involved:

### 1. Code Contribution 💻

If you are familiar with C# and the Godot Engine, pull requests are always welcome! Check our Issue Tracker for open tasks. Whether you're interested in squashing bugs, implementing new gameplay mechanics, or optimizing our `O(1)` amortized rendering architecture, your code is highly appreciated.

### 2. Content Contribution 🎵

A rhythm game is absolutely nothing without amazing music and beatmaps! You don't need to know how to code to leave a massive impact on the project:

- **Charting:** Create dynamic, brain-teasing beatmaps utilizing our multi-window, slider, and timeline rewind mechanics.
- **Original Music & Art:** Submit your original tracks, background art, and UI illustrations to be officially featured in the game.

## 🛠️ Building from Source

To compile and run the project locally on your machine:

1. Download **[Godot Engine 3.x (Mono Version)](https://godotengine.org/download/3.x)** (Testing and development is primarily done on Godot 3.6 Mono).
2. Ensure you have the required `.NET SDK` installed on your machine.
3. Clone the repository:
   ```bash
   git clone https://github.com/Nekitori17/Winithm.git
   ```
4. Open the Godot Project Manager, click **Import**, and browse to the `project.godot` file inside the cloned folder.
5. In the top-right corner of the editor, click the **Build** button to compile the C# assemblies.
6. Press `F5` or click **Play** to run the game!
